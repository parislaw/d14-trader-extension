Prop Trader Extension Icons

Generate PNG icons for the Chrome extension with light theme styling

Instructions:

  1. Click "Generate All Icons" button above
  2. Four PNG files will download to your Downloads folder
  3. Move these files to the icons/ directory of your extension
  4. The files will be named: icon16.png, icon32.png, icon48.png, icon128.png
  5. Your Chrome extension will now have proper icons!

Icon Design:

The icons feature a "14" representing the 14-day streak goal with progress dots at the bottom, using the extension's new light theme colors (white background with green accent).
